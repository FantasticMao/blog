## 关于我

我，一个 95 年、黑头发、戴眼镜的小伙子，梦想是写一个躺着就能赚钱的独立游戏 / 开源项目 / C 端产品，然后回到故乡小镇开一家自己驻唱的烧烤店。

2016 年参加工作，以 Java 谋生，也对 C、Go、JavaScript、Python 感兴趣，想要学习嵌入式、分布式、云原生领域的技术知识，坐标浙江杭州。

讨论技术可以通过文章底部的评论，或者通过首页的社交链接与我联系，随时欢迎 :)
